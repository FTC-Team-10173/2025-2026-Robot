# Team 10173 | 2025-2026 Robot Code

This repository contains the robot control code for Team 10173's 2025-2026 season.  
Key features and tools include:

- **Roadrunner**: Advanced motion planning and trajectory following.
- **FTCLib**: Library for streamlined FTC robotics programming.
- **GoBilda Odometry Pods**: High-precision tracking of robot position.
- **Limelight 3A**: Vision system for detecting targets and guiding autonomous actions.
